add_touch_handler(some_node, standard = true, priority = 0) #=> some_node click to toggle source

Adds a touch handler. By default it’s a standard handler, that means the object should respond to touches_(began|cancelled|ended). If it’s not a standard handler, it’s assumed to be a targeted handler. In that case, the object should respond to touch_(began|cancelled|ended).

In all cases, some_node MUST BE a CocosNode subclass.
