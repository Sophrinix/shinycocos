become_chipmunk_stepper() click to toggle source

Will set the node as the stepper for chipmunk (will run the chipmunk_step: selector).

The default stepper iterates only over the active shapes, not the static ones. Also, when becoming the stepper, it must have an instance variable named +@space+.

Currently, the code for the stepper is: 

  int steps = 1, i;
  cpFloat dt = 1.0f/60.0f/(cpFloat)steps;

  for(i=0; i<steps; i++){
      cpSpaceStep(space, dt);
  }
  cpSpaceHashEach(space->activeShapes, &eachShape, nil);

You can read a small discussion about this stepper here:

www.cocos2d-iphone.org/forum/topic/596
